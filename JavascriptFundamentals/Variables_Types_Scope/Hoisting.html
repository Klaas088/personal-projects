Hoisting

De javascript compiler comppileert de javascript code als volgt:

  1. Alle declaraties worden bekend bij de compiler
  2. Daarna wordt gekeken om de code uit te voeren
    3. Daardoor is het mogelijk dat variable al worden aangesproken voordat ze declareerd zijn, hij zet ze eerst allemaal om naar 'undifined'.
    Dit gedrag wordt 'hoisting' genoemd.
