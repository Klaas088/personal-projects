Namespaces and modules

https://www.typescriptlang.org/docs/handbook/namespaces-and-modules.html

Namespaces

Namspaces zijn bedoeld om structuur in code te steken.
Een namespace kan worden gebruikt over verschillende files (zelfde als .net), maar kan worden gecompileerd in 1 file.
Namespaces worden gebuild naar 1 javascript object in de global namespace. Zo blijft alles binnen scope.

Modules

Modules zijn bedoeld om in grote applicaties je applicatie modulair te gaan opbouwen.
Daardoor wordt gebruik gemaakt van een module loader (commonJS, require.js, systemJS)
Zaken in een module kunnen enkel gebruikt worden als ze eerst worden geexporteerd en geimporteerd
(eigen) Daardoor moet je ze gaan importeren in je code, dan wordt deze module ingeladen.
